# AI Interviewer

AI Interviewer is a local AI-powered interview question generator built using Flask, Ollama, and Llama 3.1.

## Features

* Generate interview questions for any domain
* Uses Ollama with Llama 3.1
* Flask backend API
* HTML, CSS, JavaScript frontend
* Runs locally on your machine

## Tech Stack

* Python
* Flask
* Flask-CORS
* Ollama
* Llama 3.1
* HTML
* CSS
* JavaScript

## Installation

1. Install Ollama

2. Pull the model:

   ollama pull llama3.1

3. Install dependencies:

   pip install -r requirements.txt

4. Start the backend:

   cd backend
   python app.py

5. Open frontend/index.html using Live Server

## Usage

Enter a domain such as:

* Python Developer
* Java Developer
* Data Scientist
* AI Engineer

Click "Generate Questions" to receive 5 interview questions generated by Llama 3.1.
